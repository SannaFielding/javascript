Redovisning av kursmomenten

Kmom01

Vilken utvecklingsmiljö använder du (berätta om något förändrats sen sist, om du svarat på frågan i tidigare kurser)?
Min utvecklingsmiljö har inte ändrats sen tidigare kurser. Jag kör fortfarande Windows, med texteditorn Atom och terminalen Ubuntu. Jag testar mina hemsidor i både Chrome och Firefox under arbetets gång.

Är du bekant med HTML och CSS tillsammans med JavaScript sedan tidigare?
Nej, det är jag inte. Jag har sett det innan, men aldrig tidigare skrivit det själv eller ens på riktigt försökt förstå vad det är som händer. Jag är ganska varm i kläderna när det kommer till HTML och CSS, och jag har programmerat i Java tidigare. Så jag hoppas att det inte ska kännas allt för främmande.

Gick det bra med JSFiddle/CodePen, några funderingar kring verktygen?
JSFiddle låg nere när jag gjorde min Me-sida, så jag har bara använt mig av CodePen än sålänge. Det gick bra, ganska rättfram, att bara klistra in koden. Vi har också fått se kodexempel i CodePen i kurserna innan denna, så det är bekant. Jag har hittils använt mig av pastebin för att dela med mig av min kod, men insett snabbt att det är besvärligt när koden ligger i flera olika filer och moduler. Det problemet löser ju CodePen åt mig, eftersom man klistrar in alla separata delar.

Gick det bra att komma i gång med kursmomentet, var det lagom, för litet, för stort, något som var svårt eller saknades?
Det gick bra. Jag tyckte att det var en lugn start. Kändes bra att börja med en lab för att få testa Javascript lite enskilt, innan man blandar in HTML och CSS. Det var skönt att få en grund till hemsida och sandboxen, så att jag kunde koncentrera mig på att implementera javascript-bitarna. Änsålänge är javascript enklare att använda tillsammans med HTML och CSS än vad jag tyckte python var.

Vilken är din TIL för detta kmom?
Hur lätt det är att implementera JavaScript på en hemsida. Med t.ex. PHP hade jag stora besvär med att gå från labb till att faktiskt implementera kod på en hemsida, men med JavaScript gick det mycket enklare. Jag hoppas att det är något jag lärt mig från föregående kurs och att det håller i sig.
